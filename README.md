# CSCS321A — Advanced Web Security

This repository contains my complete work for the Advanced Web Security course, including solved PortSwigger Web Security Academy labs, explanations, methodology, and screenshots.
All labs were performed ethically in a controlled environment for academic learning.

Project Overview
This project documents my learning journey through 32 PortSwigger labs, covering core web‑security vulnerabilities such as:

SQL Injection

Authentication flaws

Directory traversal

Access control

Cross‑Site Scripting (XSS)

CSRF

File upload vulnerabilities

Server‑side request forgery (SSRF)

Web cache poisoning

Host header attacks

Each lab includes:

Problem summary

- Concept explanation

- Step‑by‑step solution

- Screenshots of exploitation

- Key takeaways
